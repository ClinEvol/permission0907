大白菜系统开发
在制程工艺上，Intel从2015年到现在一直在魔改14nm工艺，10nm工艺说是今年6月份量产了，但在时间进度上确实要比台积电等公司落后了，AMD今年都出7nm的CPU和显卡了。今年5月份的投资会议上，Intel宣布了新一代制程工艺路线图，14nm工艺(对标台积电10nm)会继续充实产能，10nm工艺(对标台积电7nm)消费级产品今年年底购物季上架，服务器端明年上半年。
